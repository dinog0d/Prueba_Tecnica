# Prueba_Tecnica
Este repositorio es creado con la finalidad de compartir mi evaluación de la prueba técnica para el perfil de practicante de fabrica.

<p>"- Ejercicio 01: Para realizar este ejercicio, me base en una construcción clasica de la página de aprendizaje conocida como https://www.w3schools.com, una página que siempre ha sido un gran apoyo para aprender lógica básica de programación, la usé para encontrar sus plantillas de uso libre, de práctica y además para reforzar una buena práctica de estructuración de la página web. Así, construí el <"meta"> con la información de metadatos más relevante, la construcción del CSS de manera interna en el <"style"> y la estrucutra clásica del <"body">."
</p>

<p>"- Ejercicio 02: Para realizar este segundo ejercicio, me base en algunas prácticas encontradas en la misma página de https://www.w3schools.com, sumando ejercicios prácticos encontrados en https://lenguajejs.com/javascript/, esto con el fin de entender nuevamente los ejercicios básicos de JavaScript para el control en una prueba de HTML DOM. Utilicé el HTML del primer ejercicio para la base de la página y desde ahí realizar este ejercicio con las nuevas funcionalidades y estrcutura, la cual es: 

- Un segmento especializado en tener el input de la tarea y su botón correspondiente.
- Un segmento en el que se encontrarán las tareas.
- Cada tarea es un segmento en sí, que anida el nombre de la tarea y otro segmento que anida a su vez los botones de eliminar y de tarea completada."
</p>

<p>"- Ejercicio 03: Para realizar este ejercicio, como se mencionan los datos especificos a usar y que el resultado se obtenia por consola, realicé un archivo .js en el cual cree dos funciones, una para encontrar los promedios de las notas y otro que haría la transformación de los datos usando la misma función de promedios, para finalmente aplicar esta función sobre los datos y usando console.log para mostrar los resultados de la transformación."
</p>
